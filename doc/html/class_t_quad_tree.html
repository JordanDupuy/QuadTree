template<QuadTreeData T>
class TQuadTree< T >

Classe de QuadTree.

Cette classe permet de stocker des données de type T dans un QuadTree.

Paramètres du template
TLe type des données à stocker. T doit respecter le concept QuadTreeData.

template<QuadTreeData T>
void insert ( const T & t)
inline

Insère un élément dans le QuadTree.

Cette fonction insère un élément de données dans le QuadTree. Si l'élément est en dehors des limites du QuadTree, une exception de type std::domain_error est levée. Si l'élément est dans les limites d'un enfant, il est inséré dans cet enfant. Sinon, l'élément est ajouté à la liste des données du QuadTree.

Paramètres
tL'élément à insérer dans le QuadTree.
